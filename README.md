# React App

Desenvolvido usando o codespaces do github.

Esse site foi desenvolvido em react para o consumo da api-rest-node.

[API NODE](https://github.com/jeangondorek/api-rest-nodejs)

## Instalando projeto
- ```npm install```

## Rodando o projeto

- ```npm start```

## Criando Build para deploy
- ```npm run build```
