## Instalando projeto
- ```npm install```

## Rodando o projeto

- ```npm start```

## Criando Build para deploy
- ```npm run build```

## Clean cache
- ```npm cache clean -f```
- ```rm -rf node_modules package-lock.json```

Front end bastante simples só para utilização da api e estudos. A cada iteração deve se dar o reload no navegador para ver os efeitos.
